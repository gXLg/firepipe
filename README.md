# Simple Dynamic Parser
Simple dynamic lexer, parser and runner in python.

# Usage
There are plans to publish this package to PyPi in the future.
Meanwhile, you can clone the repo, link to it and import `sdparser`.

Documentation and detailed explanation will follow in distant future.
For now, see [<kbd>examples</kbd>](/examples).


Core features:
* Simple parser rules
* Fully serializable
* Reusable and fit for pipelining
* Support for typing, as well as typed operations
* Enturely iterative, stack based parsing and execution
